# lu_zeran_OOP_HW2

This is in class build that is supposed to illustrate the concepts of Object Oriented Programming (aka. OOP). This counter homework demonstrates the fundamental ideas of polymerization and extending class objects. A key part of this project is so recreate specific conditional counters and step counters based on the Counter class we created in class.

## Usage

Click through the different buttons to the value of Counter its respecting counter containers update via increment(), decrement(), or reset() methods.

Additionally, there are also conditional factors that the methods have to follow and are visually expressed through the specific iterations of the counter.
